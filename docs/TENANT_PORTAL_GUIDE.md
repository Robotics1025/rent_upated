## ğŸ“… Booking Process

### How to Book a Vacant Unit

**Step-by-Step:**

1. **Search & Select**
   - Browse available units
   - Filter by your preferences
   - Click "View Details" on unit you like

2. **Review Unit Details**
   - Check photos, amenities, location
   - Review pricing and lease terms
   - Verify availability dates

3. **Create Booking**
   - Click "Book Now"
   - Select move-in date
   - Specify lease duration
   - Add special requests (optional)

4. **Submit Booking Request**
   - Booking status: **PENDING**
   - Await admin confirmation
   - Receive notification when reviewed

5. **Make Payment**
   - Once booking **CONFIRMED**
   - Pay booking deposit
   - Choose payment method
   - Complete transaction

6. **Booking Confirmed**
   - Status changes to **CONFIRMED**
   - Receive confirmation email/SMS
   - Download booking receipt

7. **Tenancy Created**
   - Admin creates tenancy agreement
   - Review and accept lease terms
   - Download signed contract
   - Status: **ACTIVE TENANCY**

8. **Move In**
   - Unit status: **OCCUPIED**
   - Start monthly rent payments
   - Unit added to "My Units"
